# Etiquetas HTML - Guía de Referencia

## Estructura básica del documento

<!DOCTYPE html>
Identifica la versión de HTML que usamos (HTML5).

<html lang="es"> </html>
Indica que el documento está en formato HTML. El atributo "lang" especifica el lenguaje del contenido (es, en, etc.).

<head> </head>
Contiene metadatos e información sobre el documento que no se muestra directamente al usuario.

<meta> </meta>
Proporciona metadatos específicos sobre el documento.

<title> </title>
Define el título que aparece en la pestaña del navegador.

<body> </body>
Contiene todo el contenido visible de la página web.

---

## Elementos de contenido

ENCABEZADOS (HEADINGS)
<h1> </h1> hasta <h6> </h6>
Mientras más elevado es su número (1-6), menos importancia tiene en la jerarquía del contenido.

<p> </p>
Define un párrafo de texto.

<img src="imagen.jpg" alt="Texto alternativo" />
Para insertar imágenes.
- Atributo "src": Recibe la ruta de acceso de la imagen
- Atributo "alt": Recibe el texto alternativo que se mostrará si la imagen no carga

<a href="index.html">Texto de enlace</a>
Crea enlaces (links).
- Atributo "href": Recibe la página o URL hacia donde nos dirige el enlace

---

## Elementos semánticos

<header> </header>
Representa la parte superior de un sitio web o de un elemento.

<footer> </footer>
Representa la parte inferior de un sitio web o de un elemento.

<nav> </nav>
Agrupa un conjunto de enlaces de navegación.

<main> </main>
Contiene el contenido principal de un archivo HTML. SOLO debe haber uno por archivo.

<section> </section>
Se usa cuando el primer hijo es un heading y se introduce una nueva sección del documento.

<aside> </aside>
Contiene el contenido complementario que acompaña al contenido principal.

<div> </div>
Contenedor genérico que se usa cuando ninguno de los elementos semánticos anteriores es apropiado.

---

## Listas

<ul> </ul>
Crea una lista de elementos desordenados (unordered list).

<ol> </ol>
Crea una lista de elementos ordenados por números (ordered list).

<li> </li>
Define cada uno de los elementos de una lista.

---

## Formato de texto

<span> </span>
Contenedor en línea para modificar un elemento específico vía CSS.

<b> </b>
Texto en negrita.

<em> </em>
Texto en cursiva (énfasis).

---

## Otros elementos útiles

<!-- -->
Para hacer comentarios en el código HTML (no se muestran en el navegador).

<link rel="stylesheet" href="css/app.css">
Establece relaciones entre el archivo HTML y recursos externos (como hojas de estilo CSS).
- Atributo "href": Recibe la ruta del recurso a aplicar al documento HTML

## Formularios

<form> </form>
Buena práctica usar esta etiqueta para contener un formulario completo.

<input />
Permite ingresar datos en el formulario.

ATRIBUTOS PRINCIPALES DE <input>:

type - Define el tipo de dato a ingresar. Opciones:
  * text (por defecto)
  * tel (teléfono)
  * email
  * date (fecha)
  * time (hora)
  * datetime-local
  * password
  * radio (opción única - usar el mismo "name" para agrupar opciones relacionadas)
  * checkbox (opción múltiple - permite elegir varias opciones)
  * submit (botón de envío - usar "value" para cambiar el texto por defecto)

placeholder - Muestra un texto indicador dentro del input que desaparece al escribir.

value - Establece un texto predeterminado en el input.

readonly - Impide cambiar una respuesta predeterminada.

required - Marca un input como obligatorio para enviar el formulario.

<label for="id_del_input"> </label>
Proporciona una etiqueta descriptiva para cada input.
- Atributo "for": Permite seleccionar el input asociado al hacer clic en la etiqueta.

<fieldset> </fieldset>
Agrupa campos relacionados dentro del formulario.

<legend> </legend>
Proporciona un título o tema para cada grupo de campos (fieldset).

<textarea rows="5" cols="30"> </textarea>
Área de texto para mensajes más largos.
- Se pueden usar atributos "rows" y "cols" para definir el tamaño
- El texto predeterminado se coloca entre las etiquetas (no se usa "value")

<select> </select>
Crea un menú desplegable con opciones para elegir.

<option> </option>
Define cada una de las opciones dentro de un <select> o <datalist>.

ATRIBUTOS DE <option>:
- selected: Establece la opción predeterminada
- disabled: Desabilita una opción
- value: Define el valor que se envía al procesar el formulario

<datalist id="nombre_lista"> </datalist>
Permite seleccionar o escribir una opción de una lista predefinida.
- Requiere que el <input> tenga el atributo "list" que coincida con el "id" del datalist
- Las opciones se muestran usando el atributo "value" en las etiquetas <option>

---

EJEMPLO DE FORMULARIO COMPLETO:

<form>
    <fieldset>
        <legend>Información Personal</legend>
        
        <label for="nombre">Nombre:</label>
        <input type="text" id="nombre" placeholder="Ingresa tu nombre" required />
        
        <label for="email">Email:</label>
        <input type="email" id="email" required />
        
        <label for="mensaje">Mensaje:</label>
        <textarea id="mensaje" rows="5" cols="30"></textarea>
        
        <input type="submit" value="Enviar" />
    </fieldset>
</form>
